Platform support

win7+,mac will come soon;

Working mode

Direct link mode and Online link mode.Direct link is used for direct access of master computer and controlled computer. Online mode for devices with non-direct interconnection, servers are required to assist in penetration or forwarding

Instructions

  1. Sign up

    2. No account needed for Direct link mode; online mode can be registered by e-mail.

  2. Adding virtual device

    3. Up to 50 none expired virtual devices can be added to each account, each virtual device has a unique device name and device serial number

  3. Device binding

    4. For Online link,the controlled machineneeds to be binded with the virtual device by serial number, each serial number can only be bound to one machine at a time

  4. Link

    5. Direct link mode use the IP and local password of the controlled machine;Online mode use binding virtual divice name and online password.

Pay

Direct link mode is permanently free and unlimited to use;online mode requires back-end server support, current policy is 0.5 RMB / day
